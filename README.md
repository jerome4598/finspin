# finspin
finspin is intractive board for notes

Live ['Demo'](https://ranraj.github.io/finspin/index.html)

Tech Stack  
```
Elm 19.0
```

Run
```
  elm reactor
```  
Build
```
elm make src/Main.elm --output=public/main.js
```

Test (Mac) 
```
open public/index.html
```
